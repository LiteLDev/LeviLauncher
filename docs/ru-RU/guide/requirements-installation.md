# Требования и установка

На этой странице объясняется, что требуется LeviLauncher для правильной установки и запуска версий Minecraft Bedrock (GDK).

## Системные требования

| Элемент | Требование |
| --- | --- |
| Операционная система | Windows 10 или Windows 11 |
| Издание игры | Minecraft Bedrock Edition (GDK) |
| Лицензия | Легальная лицензионная копия, привязанная к вашей учетной записи Microsoft |
| Сеть | Необходима для загрузок, метаданных версий, тестов зеркал и проверок обновлений |

## Необходимые компоненты Windows

LeviLauncher может провести вас через установку отсутствующих компонентов при первом запуске или перед установкой.

- **Microsoft Gaming Services**
- **Microsoft GameInput**
- **WebView2 Runtime**

В зависимости от вашей настройки Windows, некоторые из них могут уже быть установлены.

## Перед установкой версии

Сначала выполните этот контрольный список:

1. Установите Minecraft Bedrock из Microsoft Store хотя бы один раз.
2. Запустите игру один раз, если установка из Store или служба Gaming Services все еще выглядят незавершенными.
3. Закройте игру перед использованием LeviLauncher для установки или управления версиями.

## Установка самого LeviLauncher

### Вариант А: Страница релизов GitHub

Используйте этот вариант, если вы хотите получить доступ к официальной странице загрузки LeviLauncher вместе с примечаниями к выпуску и контекстом изменений.

1. Откройте страницу [Релизы GitHub](https://github.com/LiteLDev/LeviLauncher/releases) для LeviLauncher.
2. Загрузите установщик.
3. Запустите его и завершите работу мастера настройки.

### Вариант Б: Зеркало Lanzou

Используйте этот вариант, если скорость загрузки с GitHub в вашем регионе низкая.

1. Откройте [Lanzou Cloud](https://levimc.lanzoue.com/b016ke39hc).
2. Используйте пароль `levi`.
3. Загрузите установщик и запустите его локально.

## Установка вашей первой управляемой версии

1. Откройте раздел **Загрузка** внутри LeviLauncher.
2. Выберите **Release** (Стабильная) или **Preview** (Предварительная) для нужной сборки Minecraft.
3. Выберите запись версии.
4. Решите, включать ли изоляцию.
5. Начните установку и дождитесь завершения.

## Рекомендуемая стратегия установки

### Используйте версию Minecraft Release, когда

- вам нужна самая стабильная настройка для повседневной игры
- вы создаете мир для долгосрочной игры
- вы хотите меньше сюрпризов с модами и паками

### Используйте версию Minecraft Preview, когда

- вы хотите протестировать предстоящие функции
- вы готовы к нестабильности или нарушениям совместимости
- вы намеренно держите контент Minecraft Preview отдельно от вашей основной игровой настройки

::: tip Рекомендуется для большинства игроков
Начните с одной изолированной версии **Minecraft Release**. Добавляйте версию **Minecraft Preview** позже только в том случае, если она вам конкретно нужна.
:::

## Если установка не может продолжиться

Перейдите к разделу [Обновление и устранение неполадок](./ru-RU/update-troubleshooting), если вы столкнулись с:

- отсутствием службы Gaming Services
- отсутствием службы GameInput
- отсутствием права владения в Store или состояния установки
- проблемами с правами на запись
- неудачными загрузками или проблемами с зеркалами
