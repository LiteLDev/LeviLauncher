# ワールドツール

LeviLauncher には、ワールドデータの保護や確認に役立つツールがいくつか用意されています。

## 主な機能

- ワールドを `.mcworld` などのバックアップパッケージとしてエクスポート
- ワールド関連情報の閲覧
- `level.dat` フィールドの一部を編集
- ランチャーのプロセスを通じて、より安全にワールド名を変更

## ワールドをバックアップすべきタイミング

以下のシナリオでは、必ず事前にバックアップすることをお勧めします：

- Minecraft プレビュー版（Preview）をテストする前
- 複数の Mod をインストールまたは更新する前
- 重要なワールドメタデータを修正する前
- 大幅に異なる環境間でワールドを移行する前

## `level.dat` の編集について

LeviLauncher は、`level.dat` 内の一部のフィールドを、より扱いやすい形式で表示・編集できるようにします。

::: warning 編集は慎重に行ってください
`level.dat` は機密性の高いワールドメタデータです。必ずバックアップを取り、理解しているフィールドのみを変更してください。
:::

## 推奨されるバックアップの習慣

1. 大型アップデートの前にはバックアップを取る。
2. 新しい Mod やリソースパックを導入する前にはバックアップを取る。
3. 少なくとも 1 つは、現在のバージョンのワークスペース外にバックアップを保存しておく。
