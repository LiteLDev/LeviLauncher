# Настройки и персонализация

LeviLauncher предоставляет обширный раздел настроек, позволяющий настроить пути, внешний вид, управление компонентами, параметры конфиденциальности и поведение обновлений.

## Основные разделы настроек

Приложение группирует настройки в несколько разделов, включая:

- Общие (General)
- Персонализация (Personalization)
- Компоненты (Components)
- Прочее (Others)
- Конфиденциальность (Privacy)
- Обновления (Updates)
- О программе (About)

## Что можно настроить

### Пути и хранилище

- Базовый путь к контенту
- Путь с правами на запись для управляемых файлов
- Размещение среды в соответствии со структурой вашего диска

### Внешний вид

- Язык интерфейса
- Предпочтения по теме (светлая или темная)
- Поведение цветовой темы
- Параметры фона и визуальной персонализации

### Компоненты

- Статус компонентов, связанных с GDK
- Статус или обновления языковых пакетов интерфейса (LIP)
- Другие ресурсы поддержки, управляемые лаунчером

### Конфиденциальность

- Параметры, связанные с аналитикой (если предусмотрены приложением)
- Ссылки на внешние политики конфиденциальности или условия использования

### Обновления

- Проверка наличия обновлений
- Предпочтения по каналу обновлений (например, бета-версии)

## Рекомендуемые значения по умолчанию

- Храните управляемые файлы на диске с правами на запись и достаточным свободным местом
- Используйте стабильный канал обновлений, если вам не нужны функции бета-версий
- Меняйте параметры внешнего вида только после успешной настройки базовой конфигурации

## Когда следует проверять настройки

Проверяйте настройки, когда вы:

- Перемещаете установку на другой диск
- Сталкиваетесь с ошибками прав на запись
- Хотите настроить индивидуальную персонализацию для разных устройств
- Нужно устранить неполадки с загрузкой или обнаружением компонентов
