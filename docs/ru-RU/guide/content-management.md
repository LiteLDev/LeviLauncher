# Управление контентом

LeviLauncher позволяет управлять контентом, принадлежащим конкретной версии, особенно при использовании изолированных настроек.

## Области контента, доступные в лаунчере

- миры (worlds)
- ресурс-паки (resource packs)
- поведенческие паки (behavior packs)
- паки скинов (skin packs)
- скриншоты (screenshots)
- серверы (servers)

## Почему управление контентом важно

Для многих игроков лаунчер становится наиболее полезным после установки версий, так как он помогает поддерживать порядок в различных игровых окружениях.

Примеры:

- одна версия для ванильных миров на выживание
- одна версия для тестирования ресурс-паков
- одна версия Minecraft Preview с отдельными скриншотами и записями серверов

## Общие действия

### Открытие центра контента

Откройте раздел **Контент** в приложении, чтобы увидеть количество элементов и ярлыки для основных категорий контента.

### Импорт контента

LeviLauncher поддерживает workflows импорта, такие как перетаскивание (drag-and-drop) для выбранных типов контента.

### Открытие системной папки

Используйте действия по открытию папок, когда требуется ручная проверка или очистка файлов.

### Разделение контента с помощью изоляции

Когда версия изолирована, её контент остается внутри рабочего пространства этой версии, что идеально подходит для тестирования и организации.

## Рекомендуемые привычки организации

- держите тестовые паки отдельно от основной настройки для выживания
- создавайте резервные копии важных миров перед внесением серьезных изменений
- используйте понятные имена версий, чтобы всегда знать, какому контенту где место

## Скриншоты и серверы

Скриншоты и списки серверов также легче организовать, когда каждая версия имеет отдельное назначение.
