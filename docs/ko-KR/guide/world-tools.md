# 월드 도구

LeviLauncher는 월드 데이터를 보호하고 확인하는 데 도움이 되는 몇 가지 도구를 제공합니다.

## 주요 기능

- 월드를 `.mcworld` 등의 백업 패키지로 내보내기
- 월드 관련 정보 보기
- 일부 `level.dat` 필드 편집
- 런처를 통해 월드 이름을 더 안전하게 수정

## 언제 월드를 백업해야 하나요?

다음과 같은 상황에서는 반드시 먼저 백업하세요:

- 마인크래프트 프리뷰(Preview) 버전 테스트 전
- 여러 개의 Mods 설치 또는 업데이트 전
- 중요한 월드 메타데이터를 수정하기 전
- 호환성이 크게 다른 환경 간에 월드를 이전하기 전

## `level.dat` 편집에 관하여

LeviLauncher는 `level.dat` 파일 내 일부 필드를 더 쉽게 처리할 수 있도록 노출합니다.

::: warning 신중히 편집하세요
`level.dat`는 민감한 월드 메타데이터입니다. 반드시 백업을 먼저 수행하고, 자신이 이해하는 필드만 수정하세요.
:::

## 권장 백업 습관

1. 주요 버전 업데이트 전에 백업하세요.
2. 새로운 Mods나 리소스팩을 추가하기 전에 백업하세요.
3. 현재 버전 작업 디렉터리 외부에 최소한 하나의 백업본을 유지하세요.
