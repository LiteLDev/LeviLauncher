# 자주 묻는 질문, 커뮤니티 및 피드백

## 자주 묻는 질문

### Minecraft Bedrock 정품을 반드시 소유해야 하나요?

네. LeviLauncher는 **정품 Minecraft Bedrock Edition (GDK)** 와 함께 사용하도록 설계되었습니다.

### macOS 또는 Linux에서 사용할 수 있나요?

이 사용자 문서는 **Windows**를 지원 대상으로 합니다.

### Release와 Preview 중 무엇을 선택해야 하나요?

일상적인 플레이는 Minecraft **정식판(Release)** 을 우선 선택하세요. 명확하게 다가올 콘텐츠를 체험해야 하고, 불안정성을 감수할 수 있을 때만 **프리뷰판(Preview)** 을 고려하세요.

### 격리는 필수인가요?

필수는 아니지만, 테스트, Minecraft 프리뷰(Preview) 환경 및 모드 환경에서는 매우 권장됩니다.

### 런처 내에서 월드와 리소스 팩을 관리할 수 있나요?

가능합니다. LeviLauncher는 월드, 리소스 팩, 행동 팩, 스킨 팩, 스크린샷 및 서버 목록 등의 콘텐츠 뷰를 제공합니다.

### 모드를 설치할 수 있나요?

가능하지만, 이는 고급 작업 흐름에 해당합니다. 깨끗한 격리된 버전부터 시작하여 점진적으로 테스트하는 것을 권장합니다.

## 문제 및 기능 요청을 피드백하는 방법

다음을 위해 [GitHub Issues](https://github.com/LiteLDev/LeviLauncher/issues)를 사용하여 피드백해 주세요:

- 재현 가능한 버그
- LeviLauncher와 명확하게 관련된 설치 문제
- 문서 표현이 불명확한 부분에 대한 개선 제안
- 명확한 사용 시나리오가 있는 기능 요청

## Issue를 제출하기 전에 먼저 수행할 사항

- 새로 추가한 모드 또는 리소스 팩을 일시적으로 비활성화
