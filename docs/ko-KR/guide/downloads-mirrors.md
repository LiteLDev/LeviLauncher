# 다운로드 및 미러

LeviLauncher는 미러 선택, 작업 진행률 및 로컬 패키지 가져오기 기능을 제공하여 더 안정적으로 설치를 완료할 수 있도록 도와줍니다.

## 이 영역은 무엇을 위한 것인가요?

- 설치 가능한 버전 찾아보기
- 속도 테스트 또는 미러 소스 전환
- 다운로드 및 설치 진행률 보기
- 실패한 작업을 더 쉽게 재시도
- 적절한 상황에서 로컬 설치 패키지 가져오기

## 권장 작업 흐름

1. **Download**를 엽니다.
2. 설치하려는 Minecraft **Release** 또는 **Preview**를 선택합니다.
3. LeviLauncher가 자동으로 속도를 측정하도록 하거나, 직접 적절한 미러를 선택합니다.
4. 설치 작업을 시작합니다.
5. 더 자세한 진행률을 보려면 작업 또는 다운로드 관리 뷰로 이동합니다.

## 언제 미러를 전환해야 하나요?

다음과 같은 경우 소스를 변경하는 것이 좋습니다:

- 다운로드 속도가 명백하게 이상할 때
- 특정 미러에서 연속적으로 연결에 실패할 때
- 한 소스에서 작업이 오랫동안 멈춰 있고, 다른 소스가 더 안정적일 때

## 로컬 패키지 가져오기

이미 호환되는 로컬 설치 패키지나 소스를 보유하고 있다면, LeviLauncher는 지원되는 프로세스에서 로컬 가져오기를 사용할 수 있습니다.

이는 다음 시나리오에서 매우 유용합니다:

- 네트워크가 불안정할 때
- 이전에 다운로드한 리소스를 재사용하고 싶을 때
- 더 제어 가능하고 고정된 설치 소스가 필요할 때

## 작업이 실패하면

재시도하기 전에 다음을 확인하세요:

- 디스크 공간이 충분한지
- 대상 경로에 쓰기 권한이 있는지
- 필요한 Windows 구성 요소가 설치되어 있는지
- 현재 선택한 미러가 여전히 사용 가능한지

그래도 실패하면 [업데이트 및 문제 해결](./update-troubleshooting)을 계속 참조하세요.
