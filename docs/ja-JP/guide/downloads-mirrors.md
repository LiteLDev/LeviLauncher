# ダウンロードとミラー

LeviLauncher は、ミラーの選択、タスクの進捗、ローカルパッケージのインポートなどの機能を提供し、より安定したインストールの実現をサポートします。

## このセクションの用途

- インストール可能なバージョンの閲覧
- スピードテストまたはミラーソースの切り替え
- ダウンロードおよびインストールの進捗の確認
- 失敗したタスクの再試行の容易化
- 適切な場面でのローカルインストールパッケージのインポート

## 推奨される操作フロー

1. **Download** を開く。
2. インストールする Minecraft の **Release** または **Preview** を選択する。
3. LeviLauncher に自動でスピードテストを行わせるか、手動で適切なミラーを選択する。
4. インストールタスクを開始する。
5. より詳細な進捗が必要な場合は、タスクまたはダウンロード管理ビューで確認する。

## ミラーを切り替えるべきタイミング

以下の場合は、ソースの変更をお勧めします：

- ダウンロード速度が明らかに異常な場合
- 特定のミラーへの接続が連続して失敗する場合
- タスクがあるソースで長時間停止しており、他のソースの方が安定している場合

## ローカルパッケージのインポート

互換性のあるローカルインストールパッケージやソースを既に所有している場合、LeviLauncher はサポートされているフローにおいてローカルインポートを使用できます。

これは以下の場面で非常に役立ちます：

- ネットワークが不安定な場合
- 以前にダウンロードしたリソースを再利用したい場合
- より制御可能で固定されたインストールソースが必要な場合

## タスクが失敗した場合

再試行する前に以下を確認してください：

- ディスク容量が十分にあるか
- 対象パスが書き込み可能か
- 必要な Windows コンポーネントがインストールされているか
- 現在選択しているミラーが利用可能か

それでも失敗する場合は、[更新とトラブルシューティング](./update-troubleshooting) をご覧ください。
