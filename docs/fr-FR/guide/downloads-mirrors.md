# Téléchargements et miroirs

LeviLauncher offre des options de miroir, une progression des tâches et la possibilité d'importer des paquets locaux pour vous aider à effectuer des installations plus stables.

## À quoi sert cette section ?

- Parcourir les versions installables
- Tester la vitesse ou changer de source miroir
- Voir la progression des téléchargements et des installations
- Réessayer plus facilement les tâches ayant échoué
- Importer un paquet d'installation local dans les scénarios appropriés

## Procédure recommandée

1. Ouvrez **Download**.
2. Sélectionnez la version **Release** ou **Preview** de Minecraft que vous souhaitez installer.
3. Laissez LeviLauncher tester automatiquement la vitesse, ou choisissez manuellement un miroir adapté.
4. Lancez la tâche d'installation.
5. Pour plus de détails sur la progression, rendez-vous dans la vue des tâches ou de la gestion des téléchargements.

## Quand faut-il changer de miroir ?

Il est conseillé de changer de source dans les cas suivants :

- La vitesse de téléchargement est anormalement lente
- Un miroir échoue à se connecter de manière répétée
- La tâche reste bloquée sur une source alors qu'une autre est plus stable

## Importation de paquets locaux

Si vous possédez déjà un paquet d'installation local compatible, LeviLauncher peut l'utiliser dans les flux pris en charge.

Cela est très utile dans les situations suivantes :

- Connexion réseau instable
- Vous souhaitez réutiliser des ressources déjà téléchargées
- Vous avez besoin d'une source d'installation plus contrôlable et fixe

## En cas d'échec de la tâche

Avant de réessayer, assurez-vous que :

- L'espace disque est suffisant
- Le chemin de destination est accessible en écriture
- Les composants Windows requis sont installés
- Le miroir sélectionné est toujours disponible

Si le problème persiste, consultez la page [Mise à jour et résolution de problèmes](./update-troubleshooting).
