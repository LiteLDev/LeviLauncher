# Preguntas frecuentes, comunidad y comentarios

## Preguntas frecuentes

### ¿Es obligatorio tener la versión original de Minecraft Bedrock?

Sí. LeviLauncher está diseñado para usarse junto con **Minecraft Bedrock Edition (GDK) original**.

### ¿Se puede usar en macOS o Linux?

La documentación de usuario está dirigida exclusivamente a **Windows**.

### ¿Debería elegir Release o Preview?

Para el juego diario, prioriza la **versión oficial (Release)** de Minecraft. Solo considera la **versión de vista previa (Preview)** si necesitas explícitamente probar contenido futuro y aceptas la inestabilidad.

### ¿Es obligatorio el aislamiento?

No es obligatorio, pero es muy recomendable para entornos de prueba, entornos de Minecraft de vista previa (Preview) y entornos con Mods.

### ¿Se pueden gestionar mundos y paquetes de recursos en el lanzador?

Sí. LeviLauncher proporciona vistas de contenido para mundos, paquetes de recursos, paquetes de comportamiento, paquetes de skins, capturas de pantalla y listas de servidores.

### ¿Se pueden instalar Mods?

Sí, pero se trata de un flujo de trabajo avanzado. Se recomienda empezar con una versión aislada limpia y probar gradualmente.

## Cómo informar de problemas y sugerencias de funciones

Utiliza [GitHub Issues](https://github.com/LiteLDev/LeviLauncher/issues) para informar de:

- Errores reproducibles
- Problemas de instalación claramente relacionados con LeviLauncher
- Sugerencias de mejora para la claridad de la documentación
- Solicitudes de funciones con un caso de uso claro

## Antes de enviar un Issue, se recomienda

- Actualizar a la última versión disponible
- Realizar una prueba de comparación con una versión aislada limpia
- Desactivar temporalmente los Mods o paquetes de recursos recién añadidos
- Registrar los pasos exactos que provocan el problema
