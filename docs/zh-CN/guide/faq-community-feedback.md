# 常见问题、社区与反馈

## 常见问题

### 我必须拥有 Minecraft Bedrock 正版吗？

是的。LeviLauncher 设计目标是配合**正版 Minecraft Bedrock Edition (GDK)** 使用。

### 能在 macOS 或 Linux 上使用吗？

本套用户文档面向的支持目标是 **Windows**。

### 我应该选 Release 还是 Preview？

日常游玩优先选 Minecraft **正式版（Release）**。只有你明确需要体验即将到来的内容，并且能接受不稳定时，再考虑 **预览版（Preview）**。

### 隔离是必须的吗？

不是必须，但对测试、Minecraft 预览版（Preview）环境和 Mod 环境来说非常推荐。

### 能在启动器里管理世界和资源包吗？

可以。LeviLauncher 提供世界、资源包、行为包、皮肤包、截图与服务器列表等内容视图。

### 能安装 Mods 吗？

可以，但这属于高级工作流。建议先从一个干净的隔离版本开始，逐步测试。

## 如何反馈问题与功能建议

请使用 [GitHub Issues](https://github.com/LiteLDev/LeviLauncher/issues) 反馈：

- 可复现的 Bug

## 提交 Issue 之前建议先做

- 更新到最新可用版本
- 用一个干净的隔离版本做一次对照测试
- 暂时禁用新加入的 Mods 或资源包
- 记录导致问题出现的精确步骤
