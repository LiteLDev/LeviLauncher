# 世界工具

LeviLauncher 提供了一些協助你保護和檢查世界數據的工具。

## 主要功能

- 將世界匯出為 `.mcworld` 等備份包
- 檢視世界相關資訊
- 編輯部分 `level.dat` 欄位
- 透過啟動器流程更安全地修改世界名稱

## 什麼時候應該備份世界

以下場景均建議先進行備份：

- 測試 Minecraft 預覽版（Preview）前
- 安裝或更新多個 Mods 前
- 修改重要世界元數據前
- 在差異較大的環境之間遷移世界前

## 關於 `level.dat` 編輯

LeviLauncher 可以將 `level.dat` 中的部分欄位以更易於處理的方式呈現出來。

::: warning 請謹慎編輯
`level.dat` 屬於敏感的世界元數據。請先備份，並且只修改自己理解的欄位。
:::

## 建議備份習慣

1. 在大版本更新前進行備份。
2. 在引入新 Mods 或資源包前進行備份。
3. 至少保留一份不存放在當前版本工作區內的備份。
