::: warning Windows uniquement
LeviLauncher est conçu pour **Minecraft Bedrock Edition (GDK) sur Windows** et nécessite que vous possédiez une **licence officielle**.
:::

::: tip Conseil pour les nouveaux utilisateurs
Commencez par lire la [prise en main rapide](/fr-FR/guide/quick-start), puis poursuivez avec la page [Exigences système et installation](/fr-FR/guide/requirements-installation).
:::

## Contenu de ce site

Cette documentation s'adresse aux **utilisateurs finaux**, et non aux mainteneurs ou développeurs.

- Comment installer LeviLauncher et préparer les composants préalables sous Windows
- Comment télécharger, installer et lancer Minecraft Release ou Preview
- Comment comprendre et utiliser l'isolation des versions
- Comment gérer le contenu, les mondes et les Mods
- Comment résoudre les problèmes liés aux mises à jour, aux composants manquants et aux erreurs de lancement courantes
