# 設定與個人化

LeviLauncher 提供完善的設定頁面，可用於調整路徑、外觀、組件管理、私隱偏好與更新策略。

## 主要設定分區

應用程式目前將設定分為多個區域，例如：

- General（一般）
- Personalization（個人化）
- Components（組件）
- Others（其他）
- Privacy（私隱）
- Updates（更新）
- About（關於）

## 你可以調整什麼

### 路徑與儲存

- 基礎內容路徑
- 託管檔案所在的可寫入位置
- 更適合你磁碟佈局的環境放置方式

### 外觀

- 語言
- 淺色/深色主題偏好
- 主題色行為
- 背景與視覺個人化選項

### 組件

- GDK 相關組件狀態
- LIP 狀態與更新
- 其他由啟動器維護的輔助資源

### 私隱

- 應用程式提供的分析數據（Analytics）相關選項
- 外部私隱政策與條款連結

### 更新

- 更新檢查
- Beta 更新通道偏好

## 建議預設值

- 將託管檔案存放在可寫入且空間充足的磁碟位置
- 如果你不主動追蹤 Beta 版本，優先選擇穩定更新通道
- 先確保基礎環境穩定，再微調外觀設定

## 什麼時候值得重新檢查設定

當你遇到以下情況時，建議返回設定頁面檢視一遍：

- 想把安裝遷移至另一塊磁碟
- 遇到寫入權限錯誤
- 想在不同機器上使用不同的外觀或路徑策略
- 需要排查下載或組件識別問題
