# 設定と個人化

LeviLauncher は、パス、外観、コンポーネント管理、プライバシー設定、更新戦略を調整するための、比較的包括的な設定ページを提供しています。

## 主な設定セクション

現在、アプリケーションは設定を以下のセクションに分類しています：

- General
- Personalization
- Components
- Others
- Privacy
- Updates
- About

## 調整できる項目

### パスとストレージ

- 基本コンテンツパス
- ホストファイルが置かれる書き込み可能な場所
- あなたのディスク構成により適した環境配置

### 外観

- 言語
- ライト/ダークテーマのプリファレンス
- テーマカラーの動作
- 背景および視覚的な個人化項目

### コンポーネント

- GDK 関連コンポーネントのステータス
- LIP のステータスと更新
- ランチャーによって管理されるその他の補助リソース

### プライバシー

- アプリケーションが提供する分析関連の選択
- 外部プライバシーポリシーおよび利用規約へのリンク

### 更新

- 更新チェック
- ベータ更新チャンネルのプリファレンス

## 推奨されるデフォルト値

- ホストファイルは、書き込み可能で空き容量が十分にあるディスクの場所に置く
- ベータを積極的に追わない場合は、安定した更新チャンネルを優先する
- 外観を細かく調整する前に、まず基本環境を安定させる

## 設定を再確認する価値があるタイミング

以下の状況に遭遇した場合は、設定ページを確認することをお勧めします：

- インストールを別のディスクに移行したい場合
- 書き込み権限のエラーが発生した場合
- 異なるマシンで異なる外観やパス戦略を使用したい場合
- ダウンロードまたはコンポーネントの識別に関する問題をトラブルシューティングする必要がある場合
