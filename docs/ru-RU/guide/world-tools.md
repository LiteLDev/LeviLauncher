# Инструменты для миров

LeviLauncher включает утилиты, которые помогают защищать и проверять ваши миры.

## Основные инструменты для миров

- экспорт мира в виде резервной копии, например `.mcworld`
- проверка информации, связанной с миром
- редактирование выбранных полей `level.dat`
- более безопасное переименование мира через рабочий процесс лаунчера

## Когда делать резервную копию мира

Всегда создавайте резервную копию перед тем, как:

- тестировать версии Minecraft Preview
- устанавливать или обновлять несколько модов
- изменять важные метаданные мира
- перемещать мир между очень разными настройками

## Редактирование `level.dat`

LeviLauncher может предоставить доступ к выбранным полям из `level.dat` для более удобного редактирования.

::: warning Редактируйте осторожно
`level.dat` содержит чувствительные метаданные мира. Сначала сделайте резервную копию и изменяйте только то, что вы понимаете.
:::

## Рекомендуемый порядок резервного копирования

1. Делайте резервную копию перед крупными обновлениями.
2. Делайте резервную копию перед внедрением новых модов или паков.
3. Храните хотя бы одну резервную копию вне рабочего пространства версии.
