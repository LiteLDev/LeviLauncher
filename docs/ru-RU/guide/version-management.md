# Управление версиями

LeviLauncher рассматривает каждую установленную настройку как версию, которой можно управлять независимо.

## Что вы можете делать

- устанавливать сборки Minecraft Release или Preview
- давать версиям понятные имена
- быстро запускать выбранную версию
- удалять версии, которые вам больше не нужны
- открывать настройки конкретной версии
- создавать ярлык на рабочем столе для версии

## Minecraft Release против Preview

| Тип | Лучше всего подходит для | Примечания |
| --- | --- | --- |
| Release | Стабильной ежедневной игры | Рекомендуется для большинства игроков |
| Preview | Тестирования предстоящих изменений | Чаще может повреждать миры, паки или моды |

## Объяснение изоляции

Когда изоляция включена, LeviLauncher хранит управляемые игровые данные в собственном рабочем пространстве этой версии, вместо того чтобы позволять всему использовать одно и то же расположение данных по умолчанию.

Изоляция полезна, если вы хотите:

- разные наборы контента для разных версий
- более безопасное тестирование модов или сборок Minecraft Preview
- меньший риск влияния одной настройки на другую

## Общие задачи управления версиями

### Установка новой версии

Используйте страницу **Загрузка**, затем вернитесь на главную страницу (**Home**) после завершения установки.

### Переименование версии

Используйте раздел **Версии** или страницу настроек версии, чтобы дать настройке имя, отражающее её назначение, например `Release Выживание` или `Preview Тест`.

### Удаление версии

Удаляйте версии, которые вам больше не нужны, особенно тестовые установки Minecraft Preview, которые больше не актуальны.

::: warning Будьте осторожны перед удалением
Удаление версии может удалить управляемую среду для этой настройки. Сначала создайте резервную копию важных миров.
:::

### Создание ярлыка на рабочем столе

LeviLauncher может создать ярлык на рабочем столе для конкретной версии, чтобы вы могли запускать её напрямую.

## Рекомендуемая стратегия управления версиями

- одна изолированная версия Minecraft Release для обычной игры
- одна изолированная версия Minecraft Preview только при необходимости
- одна временная тестовая версия для модов или устранения неполадок
