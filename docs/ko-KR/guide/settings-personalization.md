# 설정 및 개인화

LeviLauncher는 경로, 외관, 구성 요소 관리, 개인정보 보호 설정 및 업데이트 전략을 조정할 수 있는 비교적 완전한 설정 페이지를 제공합니다.

## 주요 설정 구역

앱은 현재 설정을 여러 영역으로 나눕니다. 예:

- General
- Personalization
- Components
- Others
- Privacy
- Updates
- About

## 무엇을 조정할 수 있나요?

### 경로 및 저장소

- 기본 콘텐츠 경로
- 호스팅 파일이 위치할 쓰기 가능한 위치
- 디스크 레이아웃에 더 적합한 환경 배치 방식

### 외관

- 언어
- 다크/라이트 테마 선호도
- 테마 색상 동작
- 배경 및 시각적 개인화 항목

### 구성 요소

- GDK 관련 구성 요소 상태
- LIP 상태 및 업데이트
- 런처가 유지 관리하는 기타 보조 리소스

### 개인정보 보호

- 애플리케이션이 제공하는 분석 관련 선택 사항
- 외부 개인정보 처리방침 및 약관 링크

### 업데이트

- 업데이트 확인
- 베타 업데이트 채널 선호도

## 권장 기본값

- 호스팅 파일을 쓰기 가능하고 공간이 충분한 디스크 위치에 배치
- 베타를 적극적으로 추적하지 않는 경우, 안정적인 업데이트 채널을 우선 유지
- 기본 환경을 안정화시킨 후 외관을 세부적으로 조정

## 설정을 다시 확인할 가치가 있는 경우

다음과 같은 상황을 겪을 때 설정 페이지를 다시 확인하는 것이 좋습니다:

- 설치를 다른 디스크로 이전하고 싶을 때
- 쓰기 권한 오류가 발생할 때
- 다른 컴퓨터에서 다른 외관 또는 경로 전략을 사용하고 싶을 때
- 다운로드 또는 구성 요소 인식 문제를 해결해야 할 때
