# 快速開始

如果你只想以最短路徑從下載進入遊戲，請先閱覽此頁。

## 1. 安裝 LeviLauncher

- 從 [GitHub Releases](https://github.com/LiteLDev/LeviLauncher/releases) 頁面或[藍奏雲鏡像](https://levimc.lanzoue.com/b016ke39hc) 下載 LeviLauncher 安裝程式，藍奏雲密碼為 `levi`。
- 在 Windows 上執行安裝程式。
- 如果安裝過程提示需要 WebView2 或其他依賴項，請允許其繼續安裝。

## 2. 確認 Microsoft Store 環境已準備就緒

- 登入擁有 **Minecraft Bedrock Edition** 的 Microsoft 帳號。
- 至少在此電腦上從 Microsoft Store 安裝過一次遊戲。
- 確保 **Gaming Services** 可用。

::: warning 必須擁有正版
LeviLauncher 用於管理 Minecraft Bedrock (GDK)，並不能替代遊戲授權。
:::

## 3. 開啟下載頁面

- 啟動 LeviLauncher。
- 從側邊欄進入 **Download**。
- 選擇 **Release** 或 **Preview**。
- 這裡的 **Release** 和 **Preview** 分別指 Minecraft 的正式版與預覽版。
- 選擇版本並開始安裝。

## 4. 決定是否啟用隔離

隔離功能會將該版本的遊戲數據存放於獨立的工作區，避免與預設的 AppData 環境混用。

以下場景建議啟用隔離：

- 測試 Mods，但不想影響主環境
- 希望將 Minecraft 正式版與預覽版分開
- 想為不同遊玩目的保留不同的世界和資源包

## 5. 啟動已安裝版本

- 返回 **Home**。
- 選取剛安裝的版本。
- 點擊 **Launch**。

如果 LeviLauncher 報告缺少 GameInput 或 Gaming Services，請先按提示完成依賴項安裝。
