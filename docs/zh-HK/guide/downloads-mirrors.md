# 下載與鏡像

LeviLauncher 提供鏡像選擇、任務進度與本地包匯入等功能，協助你更穩定地完成安裝。

## 這個區域適合做什麼

- 瀏覽可安裝版本
- 測速或切換鏡像源
- 檢視下載與安裝進度
- 更容易重試失敗任務
- 在合適場景下匯入本地安裝包

## 建議操作流程

1. 打開 **Download**。
2. 選擇你要安裝的 Minecraft **Release** 或 **Preview**。
3. 讓 LeviLauncher 自動測速，或手動選擇合適鏡像。
4. 啟動安裝任務。
5. 如需更詳細進度，進入任務或下載管理視圖檢視。

## 什麼時候應該切換鏡像

以下情況建議換源：

- 下載速度明顯異常
- 某個鏡像連續連接失敗
- 任務在某個源上長時間卡住，而其他源更穩定

## 本地包匯入

如果你已經擁有相容的本地安裝包或來源，LeviLauncher 在支援的流程下可以使用本地匯入。

這在以下場景會很有幫助：

- 網絡不穩定
- 你想複用之前下載好的資源
- 你需要更可控、更固定的安裝來源

## 如果任務失敗

重試前請先確認：

- 磁碟空間足夠
- 目標路徑可寫入
- 所需 Windows 組件已安裝
- 當前選擇的鏡像仍然可用

如果仍然失敗，請繼續閱覽[更新與故障排查](./update-troubleshooting)。
