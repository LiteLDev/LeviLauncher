# クイックスタート

ダウンロードからゲーム起動までの最短ルートを知りたい場合、まずこのページを確認してください。

## 1. LeviLauncher のインストール

- [GitHub Releases](https://github.com/LiteLDev/LeviLauncher/releases) ページまたは[藍奏雲ミラー](https://levimc.lanzoue.com/b016ke39hc)から LeviLauncher インストーラーをダウンロードします（藍奏雲のパスワードは `levi` です）。
- Windows でインストーラーを実行します。
- インストール中に WebView2 またはその他の依存関係が必要と表示された場合は、続行を許可してください。

## 2. Microsoft Store 環境の準備確認

- **Minecraft Bedrock Edition** のライセンスを持つ Microsoft アカウントにログインします。
- 現在の PC で Microsoft Store から少なくとも一度ゲームをインストール済みであることを確認します。
- **Gaming Services** が利用可能であることを確認します。

::: warning 正規版が必要です
LeviLauncher は Minecraft Bedrock (GDK) の管理に使用され、ゲームのライセンスを代替することはできません。
:::

## 3. ダウンロードページを開く

- LeviLauncher を起動します。
- サイドバーから **Download** に移動します。
- **Release** または **Preview** を選択します。
- ここで **Release** と **Preview** は Minecraft の正式版とプレビュー版を指します。
- バージョンを選択し、インストールを開始します。

## 4. 隔離の有効化の可否を決定

隔離は、そのバージョンのゲームデータを独立したワークスペースに入れ、デフォルトの AppData 環境と混在させなくなります。

以下の場面で隔離の使用を推奨します：

- Mod のテストをしたいが、メイン環境に影響を与えたくない場合
- Minecraft 正式版とプレビュー版を分離したい場合
- 遊び方ごとに異なるワールドやリソースパックを保持したい場合

## 5. インストール済みバージョンの起動

- **Home** に戻ります。
- 剛インストールしたバージョンを選択します。
- **Launch** をクリックします。

LeviLauncher が GameInput または Gaming Services の不足を報告した場合は、まずプロンプトに従って依存関係をインストールしてください。
