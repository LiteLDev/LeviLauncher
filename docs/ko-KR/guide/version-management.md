# 버전 관리

LeviLauncher는 각 설치된 환경을 독립적으로 관리할 수 있는 버전으로 간주합니다.

## 무엇을 할 수 있나요?

- 정식판 또는 프리뷰판 설치
- 버전에 식별하기 쉬운 이름 설정
- 지정된 버전을 빠르게 실행
- 더 이상 필요 없는 버전 삭제
- 버전 전용 설정 열기
- 버전용 데스크톱 바로가기 생성

## 정식판(Release)과 프리뷰판(Preview)의 차이점

| 유형 | 적합 대상 | 설명 |
| --- | --- | --- |
| Release(정식판) | 안정적인 일상 플레이 | 대부분의 플레이어에게 적합 |
| Preview(프리뷰판) | 새로운 콘텐츠 사전 테스트 | 월드, 팩 또는 모드 호환성 변경이 더 자주 발생할 수 있음 |

## 격리란 무엇인가요?

격리를 활성화하면, LeviLauncher는 해당 버전의 게임 데이터를 기본 데이터 위치와 공유하지 않고 자체 작업 영역에 저장합니다.

격리는 다음 요구 사항에 적합합니다:

- 서로 다른 버전에서 서로 다른 콘텐츠 조합 사용
- 모드 또는 Minecraft 프리뷰(Preview)를 더 안전하게 테스트
- 한 환경이 다른 환경에 영향을 미치는 위험을 줄임

## 일반적인 버전 작업

### 새 버전 설치

**Download** 페이지에서 설치를 완료하면, **Home**으로 돌아가 실행할 수 있습니다.

### 버전 이름 변경

**Versions** 또는 버전 설정 페이지에서 해당 환경에 더 쉽게 식별할 수 있는 용도 이름(예: `Release Survival` 또는 `Preview Test`)을 설정합니다.

### 버전 삭제

더 이상 필요 없는 버전, 특히 테스트가 완료된 Minecraft 프리뷰(Preview) 환경을 제거합니다.

::: warning 삭제 전에 먼저 확인하세요
버전을 삭제하면 해당 호스팅 환경의 관련 콘텐츠가 제거될 수 있습니다. 중요한 월드는 먼저 백업하세요.
:::

### 데스크톱 바로가기 생성

LeviLauncher는 지정된 버전에 대한 데스크톱 바로가기를 생성할 수 있어 직접 실행하기 편리합니다.

## 권장 버전 전략

- 일상 플레이를 위한 격리된 정식판(Release) 버전 하나
- 필요할 때만 사용하는 격리된 프리뷰판(Preview) 버전 하나
- 모드 또는 문제 해결을 위한 임시 테스트 버전 하나
