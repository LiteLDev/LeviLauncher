# Первый запуск

При первом открытии LeviLauncher приложение сосредотачивается на том, чтобы убедиться, что Windows и Minecraft готовы к управляемым установкам.

## Чего ожидать

При первом запуске LeviLauncher может:

- проверить доступность службы Gaming Services
- проверить доступность службы GameInput
- подтвердить доступ к среде Minecraft Bedrock (GDK)
- показать инструкции по первоначальной настройке, если отсутствуют необходимые компоненты

## Типичный процесс первого запуска

1. Откройте LeviLauncher.
2. Прочитайте любые подсказки по настройке, отображаемые лаунчером.
3. Установите или восстановите отсутствующие предварительные требования.
4. Вернитесь в лаунчер и повторно проверьте статус.
5. Перейдите на страницу **Загрузка**, как только требования будут выполнены.

## Если компонент отсутствует

### Gaming Services

Если служба Gaming Services отсутствует или повреждена, LeviLauncher может перенаправить вас в Microsoft Store или предложить восстановить этот компонент.

### GameInput

Если служба GameInput отсутствует, установите необходимый распространяемый пакет (redistributable), когда лаунчер запросит это.

### Minecraft Bedrock не обнаружен

Убедитесь, что вы:

- владеете игрой в текущей учетной записи Microsoft
- установили её из Microsoft Store на этом ПК
- не пытаетесь использовать LeviLauncher в качестве замены официальной процедуры лицензирования

## Хорошие настройки по умолчанию для новых пользователей

- сначала используйте одну изолированную версию Minecraft Release
- держите пути для загрузок и контента на диске с правами записи
- избегайте импорта модов, пока первый чистый запуск не пройдет успешно
