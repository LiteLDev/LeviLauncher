# 版本管理

LeviLauncher 会把每个已安装环境视为一个可独立管理的版本。

## 你可以做什么

- 安装正式版或预览版
- 为版本设置便于识别的名称
- 快速启动指定版本
- 删除不再需要的版本
- 打开版本专属设置
- 为版本创建桌面快捷方式

## 正式版（Release）与预览版（Preview）的区别

| 类型 | 适合谁 | 说明 |
| --- | --- | --- |
| Release（正式版） | 稳定日常游玩 | 适合大多数玩家 |
| Preview（预览版） | 提前测试新内容 | 更容易出现世界、包或 Mod 兼容性变化 |

## 什么是隔离

启用隔离后，LeviLauncher 会把该版本的游戏数据保存在自己的工作区中，而不是与默认数据位置共享。

隔离适合这些需求：

- 不同版本使用不同内容组合
- 更安全地测试 Mods 或 Minecraft 预览版（Preview）
- 降低一个环境影响另一个环境的风险

## 常见版本操作

### 安装新版本

在 **Download** 页面完成安装，结束后回到 **Home** 即可启动。

### 重命名版本

在 **Versions** 或版本设置页，为该环境设置更容易识别的用途名，例如 `Release Survival` 或 `Preview Test`。

### 删除版本

移除不再需要的版本，尤其是已经完成测试的 Minecraft 预览版（Preview）环境。

::: warning 删除前先确认
删除版本可能会移除该托管环境中的相关内容。重要世界请先备份。
:::

### 创建桌面快捷方式

LeviLauncher 可以为指定版本创建桌面快捷方式，方便你直接启动。

## 推荐的版本策略

- 一个隔离的正式版（Release）版本用于日常游玩
- 一个隔离的预览版（Preview）版本只在需要时使用
- 一个临时测试版本用于 Mods 或排错
