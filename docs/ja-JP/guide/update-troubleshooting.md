# 更新とトラブルシューティング

LeviLauncher のインストール、更新、起動が予期せぬ動作をする場合、まずこのページを参照してください。

## よくある問題

### Gaming Services が不足または破損している

- ランチャーが提供するガイドに従ってインストールまたは修復を行ってください。
- 必要に応じて、Microsoft Store に戻って Minecraft Bedrock のインストール状態を確認してください。

### GameInput が不足している

- LeviLauncher のプロンプトに従って必要な redistributable をインストールしてください。
- インストール後、必要に応じてランチャーを再起動してください。

### インストールパスに書き込めない

- **Settings** でホストコンテンツのパスを書き込み可能な場所に変更してください。
- ランチャーがインストールまたは自己更新中に特権の昇格を要求する場合、信頼できると確認した場合にのみ許可を実行してください。

### 特定のバージョンが起動できない

- Mods 無しでそのバージョンが起動するか、まずテストしてください。
- 必要な Windows コンポーネントが依然として存在するか確認してください。
- そのバージョンが完全にインストールされているか確認してください。
- 問題を再現するために、クリーンな隔離テスト環境を試してみてください。

### ダウンロードが遅い、または繰り返し失敗する

- ミラーを切り替えてください。
- 後でもう一度試してください。
- 条件が許せば、ローカルパッケージソースに変更してください。

## より安全な回復手順

1. まず重要なワールドをバックアップしてください。
2. 最近追加した Mods やリソースパックを削除してください。
3. クリーンな隔離バージョンを使って対照テストを行ってください。
4. 必要に応じて、影響を受けたバージョンを再インストールしてください。

## 自己更新について

LeviLauncher はアプリケーションの更新を確認、ダウンロード、インストールできます。インストールディレクトリに書き込み権限がない場合、一部の環境で管理者権限が必要になる場合があります。

## Issue を立てるタイミング

以下の条件を満たす場合、GitHub に問題を報告することを推奨します：

- 問題を安定して再現できる
- 基本的な回復手順を試した後も依然として問題が存在する
- 問題が一般的な Windows 環境問題ではなく、LeviLauncher 自体にあるように見える

報告する際は、以下の情報を添付してください：

- Windows バージョン
- LeviLauncher バージョン
- 具体的な操作手順
- スクリーンショットまたはログ（ある場合）

問題の報告先：[GitHub Issues](https://github.com/Levi-Launcher/LeviLauncher/issues)
