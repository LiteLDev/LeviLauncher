# Обновление и устранение неполадок

Используйте эту страницу, если установка, обновление или запуск LeviLauncher работают не так, как ожидалось.

## Общие проблемы

### Служба Gaming Services отсутствует или повреждена

- Восстановите или установите её через направляемый процесс.
- При необходимости повторно проверьте состояние установки Minecraft Bedrock в Microsoft Store.

### Отсутствует служба GameInput

- Установите необходимый распространяемый пакет, когда LeviLauncher запросит это.
- При необходимости перезапустите лаунчер после установки.

### Путь установки недоступен для записи

- Переместите путь управляемого контента в доступное для записи расположение в разделе **Настройки**.
- Если лаунчер запрашивает повышение прав для установки или самообновления, разрешите это, если вы доверяете действию.

### Версия не запускается

- протестируйте ту же версию без модов
- убедитесь, что необходимые компоненты Windows все еще присутствуют
- проверьте, что версия была установлена корректно
- повторите попытку с чистой изолированной тестовой настройкой

### Загрузки слишком медленные или постоянно завершаются неудачей

- переключите зеркала
- повторите попытку позже
- используйте локальный источник пакетов, если он доступен

## Безопасные шаги восстановления

1. Создайте резервную копию важных миров.
2. Удалите недавние моды или паки.
3. Протестируйте с чистой изолированной версией.
4. При необходимости переустановите затронутую версию.

## Поведение самообновления

LeviLauncher может проверять, загружать и устанавливать обновления приложения. Некоторые среды могут требовать повышенных прав доступа, если каталог установки недоступен для записи.

## Когда сообщать об ошибке

Откройте задачу на GitHub, если:

- вы можете стабильно воспроизвести проблему
- проблема сохраняется после выполнения базовых шагов восстановления
- проблема выглядит специфичной для LeviLauncher, а не общей проблемой настройки Windows

При сообщении включите:

- версию Windows
- версию LeviLauncher
- точные шаги, которые вы выполнили
- скриншоты или логи, если они доступны

Открыть задачи можно здесь: [GitHub Issues](https://github.com/LiteLDev/LeviLauncher/issues)
