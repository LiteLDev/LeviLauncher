::: warning Windows 전용
LeviLauncher는 **Windows의 Minecraft Bedrock Edition (GDK)** 용으로 설계되었으며, **정품 라이선스**가 필요합니다.
:::

::: tip 새 사용자는 여기서 시작하세요
먼저 [빠른 시작](/ko-KR/guide/quick-start)을 읽은 후 [시스템 요구 사항 및 설치](/ko-KR/guide/requirements-installation)를 계속 읽으세요.
:::

## 이 사이트는 무엇을 포함하나요?

이 문서는 **최종 사용자**를 대상으로 하며, 유지 관리자나 개발자를 대상으로 하지 않습니다.

- LeviLauncher 설치 방법 및 Windows 사전 구성 요소 준비 방법
- Minecraft 정식판 또는 프리뷰 판을 다운로드, 설치 및 실행하는 방법
- 버전 격리의 원리와 사용 방법
- 콘텐츠, 월드 및 모드 관리 방법
- 업데이트, 누락된 구성 요소 및 일반적인 실행 문제 해결 방법
