# 常見問題、社群與回饋

## 常見問題

### 我必須擁有 Minecraft Bedrock 正版嗎？

是的。LeviLauncher 的設計目標是配合**正版 Minecraft Bedrock Edition (GDK)** 使用。

### 能在 macOS 或 Linux 上使用嗎？

本套用戶文檔面向的支持目標是 **Windows**。

### 我應該選 Release 還是 Preview？

日常遊玩優先選擇 Minecraft **正式版（Release）**。只有當你明確需要體驗即將推出的內容，並且能接受不穩定性時，才考慮 **預覽版（Preview）**。

### 隔離是必須的嗎？

不是必須，但對於測試、Minecraft 預覽版（Preview）環境及 Mod 環境來說，強烈建議使用。

### 能在啟動器裡管理世界和資源包嗎？

可以。LeviLauncher 提供世界、資源包、行為包、皮膚包、截圖與伺服器列表等內容視圖。

### 能安裝 Mods 嗎？

可以，但這屬於進階工作流程。建議先從一個乾淨的隔離版本開始，逐步進行測試。

## 如何回饋問題與功能建議

請使用 [GitHub Issues](https://github.com/LiteLDev/LeviLauncher/issues) 進行回饋：

- 可重現的錯誤 (Bug)
- 與 LeviLauncher 明確相關的安裝問題
- 針對文檔表達不清的改進建議
- 帶有明確使用場景的功能請求

## 提交 Issue 之前建議先做

- 更新至最新可用版本
- 使用一個乾淨的隔離版本進行對照測試
- 暫時停用新加入的 Mods 或資源包
- 記錄導致問題出現的精確步驟
