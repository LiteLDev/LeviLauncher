# 下载与镜像

LeviLauncher 提供镜像选择、任务进度与本地包导入等能力，帮助你更稳定地完成安装。

## 这个区域适合做什么

- 浏览可安装版本
- 测速或切换镜像源
- 查看下载与安装进度
- 更容易重试失败任务
- 在合适场景下导入本地安装包

## 推荐操作流程

1. 打开 **Download**。
2. 选择你要安装的 Minecraft **Release** 或 **Preview**。
3. 让 LeviLauncher 自动测速，或手动选择合适镜像。
4. 启动安装任务。
5. 如需更详细进度，进入任务或下载管理视图查看。

## 什么时候应该切换镜像

以下情况建议换源：

- 下载速度明显异常
- 某个镜像连续连接失败
- 任务在一个源上长时间卡住，而其他源更稳定

## 本地包导入

如果你已经拥有兼容的本地安装包或来源，LeviLauncher 在支持的流程下可以使用本地导入。

这在以下场景会很有帮助：

- 网络不稳定
- 你想复用之前下载好的资源
- 你需要更可控、更固定的安装来源

## 如果任务失败

重试前先确认：

- 磁盘空间足够
- 目标路径可写
- 所需 Windows 组件已安装
- 当前选择的镜像仍然可用

如果仍然失败，请继续查看[更新与故障排查](./update-troubleshooting)。
