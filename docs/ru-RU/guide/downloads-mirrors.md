# Загрузки и зеркала

LeviLauncher включает инструменты, которые помогут вам выбрать подходящий источник загрузки, отслеживать прогресс и использовать локальные установочные файлы при необходимости.

## Для чего нужна эта область

- просмотр доступных для установки версий
- тестирование или выбор доступных зеркал
- отслеживание прогресса активной загрузки и установки
- более легкий повторный запуск неудачных задач
- импорт локального установочного пакета, если это поддерживается вашим рабочим процессом

## Рекомендуемый рабочий процесс

1. Откройте раздел **Загрузка**.
2. Выберите нужную версию Minecraft Release или Preview.
3. Позвольте LeviLauncher протестировать или использовать предпочитаемое вами зеркало.
4. Запустите задачу.
5. Откройте view задачи или диспетчер загрузок, если вам нужен более подробный прогресс.

## Когда следует менять зеркала

Рекомендуется сменить зеркало, если:

- скорость загрузки необычно низкая
- зеркало неоднократно отказывает в подключении
- задача зависает на слишком долгое время по сравнению с другим источником

## Импорт локального пакета

Если у вас уже есть совместимый локальный пакет или источник установщика, LeviLauncher может использовать потоки локального импорта, где это поддерживается.

Это полезно, когда:

- ваше сетевое соединение нестабильно
- вы хотите повторно использовать ранее загруженный пакет
- вам нужен более предсказуемый источник установки

## Если задача не выполнена

Перед повторной попыткой проверьте следующее:

- доступно ли достаточно места на диске
- доступен ли целевой путь для записи
- установлены ли необходимые компоненты Windows
- отвечает ли выбранное вами зеркало

Если проблема сохраняется, перейдите к разделу [Обновление и устранение неполадок](./update-troubleshooting).
