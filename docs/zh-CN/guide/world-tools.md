# 世界工具

LeviLauncher 提供了一些帮助你保护和检查世界数据的工具。

## 主要能力

- 把世界导出为 `.mcworld` 等备份包
- 查看世界相关信息
- 编辑部分 `level.dat` 字段
- 通过启动器流程更安全地修改世界名称

## 什么时候应该备份世界

以下场景都建议先备份：

- 测试 Preview 版本前
- 安装或更新多个 Mods 前
- 修改重要世界元数据前
- 在差异较大的环境之间迁移世界前

## 关于 `level.dat` 编辑

LeviLauncher 可以把 `level.dat` 中的部分字段以更易处理的方式暴露出来。

::: warning 请谨慎编辑
`level.dat` 属于敏感的世界元数据。请先备份，并且只修改自己理解的字段。
:::

## 推荐备份习惯

1. 大版本更新前备份。
2. 引入新 Mods 或资源包前备份。
3. 至少保留一份不放在当前版本工作区内的备份。
