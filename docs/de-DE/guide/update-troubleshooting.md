# Updates und Fehlerbehebung

Wenn die Installation, das Update oder der Start von LeviLauncher nicht wie erwartet funktioniert, konsultiere bitte zunächst diese Seite.

## Häufige Probleme

### Gaming Services fehlt oder ist beschädigt

- Verwende die im Launcher bereitgestellte Anleitung zur Installation oder Reparatur.
- Prüfe gegebenenfalls im Microsoft Store den Installationsstatus von Minecraft Bedrock.

### GameInput fehlt

- Installiere das erforderliche Redistributable, wie von LeviLauncher angezeigt.
- Starte den Launcher nach der Installation gegebenenfalls neu.

### Installationspfad ist nicht beschreibbar

- Ändere in den **Einstellungen** den Pfad für verwaltete Inhalte in einen beschreibbaren Ort.
- Wenn der Launcher beim Installieren oder Selbst-Update erhöhte Rechte verlangt, erlaube die Ausführung nur, wenn du die Quelle vertraust.

### Eine Version startet nicht

- Teste zunächst, ob die Version ohne Mods starten kann.
- Prüfe, ob die erforderlichen Windows-Komponenten noch vorhanden sind.
- Stelle sicher, dass die Version vollständig installiert ist.
- Versuche, eine saubere isolierte Testumgebung zu erstellen, um das Problem zu reproduzieren.

### Download ist langsam oder bricht ständig ab

- Wechsle den Spiegel (Mirror).
- Versuche es später erneut.
- Verwende bei Gelegenheit eine lokale Paketquelle.

## Sicherere Wiederherstellungsschritte

1. Sichere zunächst wichtige Welten.
2. Entferne zuletzt hinzugefügte Mods oder Ressourcenpakete.
3. Führe einen Vergleichstest mit einer sauberen isolierten Version durch.
4. Installiere die betroffene Version gegebenenfalls neu.

## Über das Selbst-Update

LeviLauncher kann auf Updates prüfen, diese herunterladen und installieren. Falls das Installationsverzeichnis nicht beschreibbar ist, werden unter Umständen Administratorrechte benötigt.

## Wann man ein Issue erstellen sollte

Ein Problem sollte auf GitHub gemeldet werden, wenn folgende Bedingungen erfüllt sind:

- Das Problem ist stabil reproduzierbar.
- Nach den grundlegenden Wiederherstellungsschritten besteht es weiterhin.
- Das Problem scheint eher an LeviLauncher selbst und nicht an einer allgemeinen Windows-Umgebung zu liegen.

Bitte füge bei der Meldung folgende Informationen bei:

- Windows Version
- LeviLauncher Version
- Deine konkreten Schritte zur Reproduktion
- Screenshots oder Logs (falls vorhanden)
