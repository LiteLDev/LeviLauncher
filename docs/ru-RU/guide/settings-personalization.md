# Настройки и персонализация

LeviLauncher включает обширную область настроек, позволяющую настраивать пути, внешний вид, управление компонентами, параметры конфиденциальности и поведение обновлений.

## Основные разделы настроек

Приложение группирует настройки в несколько разделов, включая:

- Общие (General)
- Персонализация (Personalization)
- Компоненты (Components)
- Прочее (Others)
- Конфиденциальность (Privacy)
- Обновления (Updates)
- О программе (About)

## Что можно настроить

### Пути и хранилище

- базовый путь к контенту
- доступное для записи расположение для управляемых файлов
- размещение среды, соответствующее структуре вашего диска

### Внешний вид

- язык
- предпочтение светлой или темной темы
- поведение цветовой темы
- параметры фона и визуальной персонализации

### Компоненты

- статус компонентов, связанных с GDK
- статус или обновления LIP
- другие ресурсы поддержки, управляемые лаунчером

### Конфиденциальность

- выбор, связанный с аналитикой (если предлагается приложением)
- внешние ссылки на политику конфиденциальности или условия использования

### Обновления

- проверки обновлений
- предпочтения канала бета-обновлений

## Рекомендуемые значения по умолчанию

- храните управляемые файлы на диске с правами записи и достаточным свободным местом
- начните со стабильного канала обновлений, если вам не нужно поведение бета-версии
- изменяйте внешний вид только после того, как ваша базовая настройка будет хорошо работать

## Когда следует проверять настройки

Проверяйте настройки, когда вы:

- перемещаете установку на другой диск
- сталкиваетесь с ошибками прав на запись
- хотите отдельную персонализацию для каждого компьютера
-需要 устранить неполадки с загрузками или обнаружением компонентов
