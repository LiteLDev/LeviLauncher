# 콘텐츠 관리

LeviLauncher는 버전 관련 콘텐츠를 관리할 수 있으며, 특히 격리된 환경과 함께 사용하기에 적합합니다.

## 런처에서 관리할 수 있는 콘텐츠 유형

- 월드
- 리소스 팩
- 행동 팩
- 스킨 팩
- 스크린샷
- 서버 목록

## 왜 이 부분이 중요한가요?

많은 플레이어가 런처를 장기간 사용하기 시작하는 진짜 이유는 "버전을 설치하는 것"이 아니라 "다른 환경을 명확히 정리하는 것"입니다.

예를 들어:

- 한 버전은 오리지널 생존 월드 전용
- 한 버전은 리소스 팩 테스트 전용
- Minecraft 프리뷰(Preview) 버전은 스크린샷과 서버 목록만 별도로 저장

## 일반적인 작업

### 콘텐츠 개요 열기

앱에서 **Content**로 이동하면 다양한 콘텐츠의 수량과 빠른 진입점을 볼 수 있습니다.

### 콘텐츠 가져오기

LeviLauncher는 일부 콘텐츠 유형에 대해 드래그 앤 드롭과 같은 가져오기 프로세스를 지원합니다.

### 실제 폴더 열기

파일을 수동으로 검사하거나 정리해야 하는 경우, 런처에서 제공하는 디렉터리 열기 작업을 사용할 수 있습니다.

### 격리를 사용하여 콘텐츠 독립 유지

버전에서 격리를 활성화하면 해당 버전의 콘텐츠가 독립된 작업 영역에 저장되며, 테스트 및 보관 정리에 매우 적합합니다.

## 권장 정리 습관

- 테스트용 리소스 팩을 메인 월드와 혼용하지 마세요.
- 중요한 변경 사항을 적용하기 전에 중요한 월드를 먼저 백업하세요.
- 버전에 명확한 이름을 지정하여 콘텐츠 소속을 혼동하지 마세요.

## 스크린샷 및 서버에 대하여

다른 버전이 서로 다른 용도를 담당한다면, 스크린샷과 서버 목록도 버전별로 분리하는 것이 일반적으로 더 명확합니다.
